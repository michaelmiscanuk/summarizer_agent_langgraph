‚öôÔ∏è How It Works

  1. User Input: You enter text in the frontend interface
  2. API Request: Frontend sends a POST request to the FastAPI backend
  3. LangGraph Processing: Backend runs a multi-node workflow:
    • Node 1: Processes input and calculates word count
    • Node 2: Generates summary and sentiment using Ollama LLM
  4. Response: Results are returned as JSON to the frontend
  5. Display: Beautiful UI displays the analysis results
